# zSkyDucky

Site oficial do canal **zSkyDucky**, criado para reunir informações sobre o streamer, seus parceiros, redes sociais e um espaço para os fãs comentarem sobre o canal.

🔗 **Acesse o site:** [zskyducky1.github.io/zSkyDucky-Page](https://zskyducky1.github.io/zSkyDucky-Page/)

---

## 📖 Sobre o projeto

O **zSkyDucky** é um canal de lives criado em 2011, integrante do clã **CLT – CELESTES**. Este site foi desenvolvido para apresentar o canal, seus parceiros, redes sociais e o conteúdo em vídeo, além de permitir que os visitantes deixem comentários e entrem em contato.

**Dias de live:** segunda, quarta e sexta às 19:00 (com tolerância de até 19:30)
**Sábado:** lives extras a partir das 15h, 16h ou 17h

---

## 🗺️ Páginas do site

| Página | Descrição |
|---|---|
| `index.html` | Página inicial com apresentação do canal, descrição e formulário de contato |
| `page2.html` | Lista dos canais parceiros com links diretos para a Twitch |
| `page3.html` | Redes sociais oficiais (Twitch, YouTube, Instagram, Facebook, TikTok, Kick) |
| `page4.html` | Vídeo de destaque do canal, informações da equipe e sessão de comentários |

---

## ✨ Funcionalidades

- 🍔 **Menu hambúrguer responsivo** para navegação em dispositivos móveis
- 💌 **Formulário de contato** com validação de nome, e-mail e telefone
- 📱 **Máscara automática de telefone** no formato `(XX) XXXXX-XXXX`
- 💬 **Sistema de comentários** com armazenamento local (`localStorage`), permitindo que os comentários persistam entre visitas
- 🎨 **Design "glassmorphism"** (efeito de vidro) com fundo animado e elementos translúcidos
- 📐 **Layout responsivo** adaptado para desktop, tablet e celular

---

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização, efeitos de glassmorphism, animações (`@keyframes`) e media queries para responsividade
- **JavaScript (Vanilla)** — interatividade do menu, máscara de telefone, validação de formulário e sistema de comentários

## 🚀 Como executar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/zskyducky1/zSkyDucky-Page.git
   ```
2. Entre na pasta do projeto:
   ```bash
   cd zSkyDucky-Page
   ```
3. Abra o arquivo `index.html` no navegador (ou use uma extensão como *Live Server* no VS Code).
